## 变量定义规则

### 通用格式变量

| 变量 | 格式 | 取值说明 |
|------|------|---------|
| `{date}` | YYYY-MM-DD | 审计日期 |
| `{timestamp}` | YYYY-MM-DD HH:MM | 报告生成时间（含时分） |
| `{asin}` | 字符串 | 亚马逊 ASIN，10位字母数字 |
| `{marketplace}` | 站点代码 | US / UK / DE / FR / JP / CA / AU / IT / ES / MX |
| `{category}` | 文字 | 产品所在类目路径，如 "Home & Kitchen > Kitchen & Dining > Faucets" |
| `{brand}` | 文字 | 品牌名，无品牌写"无品牌" |
| `{fetch_method}` | 文字 | 实际使用的数据获取方式，如 "web_fetch desktop" / "web_fetch mobile" / "agent-browser" / "用户提供" |

### 评分变量

| 变量 | 格式 | 取值说明 |
|------|------|---------|
| `{score}` | 整数 1-10 | 单维度评分，无小数 |
| `{w}` | 数字，保留1位小数 | 加权分 = score × 权重%，如 8×11% = 0.9 |
| `{total}` | 数字，保留1位小数 | 加权总分（14维度 w 之和），满分 10.0 |
| `{cdq_total}` | 数字，保留1位小数 | CDQ 总分，与 {cdq} 相同 |
| `{cdq}` | 数字，保留1位小数 | CDQ 加权总分，满分 10.0 |

### 状态标识变量

| 变量 | 格式 | 取值说明 |
|------|------|---------|
| `{s}` | emoji | ✅ 优秀（≥8）/ ⚠️ 需关注（5-7）/ ❌ 不通过（<5） |
| `{status}` | emoji | 同 {s}，用于表格"状态"列 |
| `{r}` | emoji | 同 {s}，用于 Review/FAQ 表格的"已回应"/"结果"列 |
| `{n}` | 文字 | 备注说明，简短描述（≤20字） |
| `{d}` | 文字 | 详情描述，可较长 |

### 产品信息变量

| 变量 | 格式 | 取值说明 |
|------|------|---------|
| `{title}` | 文字 | 当前产品标题全文 |
| `{current_title}` | 文字 | 同 {title}，用于"当前标题"展示 |
| `{title_chars}` | 整数 | 当前标题字符数（含空格） |
| `{chars}` | 整数 | 当前字段的字符数，上下文决定是哪个字段 |
| `{price}` | 货币格式 | 如 $29.99，含货币符号 |
| `{rating}` | 数字，保留1位 | 如 4.3 |
| `{review_count}` | 整数 | 总评论数，含逗号分隔，如 1,234 |
| `{has_hl}` | 文字 | "已填写" / "未填写" |
| `{current}` | 文字 | 当前字段的原始内容 |
| `{optimized}` | 文字 | 优化后的内容 |
| `{optimized_title}` | 文字 | 优化后的标题（≤75字符） |
| `{optimized_highlights}` | 文字 | 优化后的 Product Highlights（≤125字符） |
| `{search_terms}` | 文字 | 优化后的 Search Terms（250字节内） |
| `{breadcrumb}` | 文字 | 当前类目路径，如 "Home > Kitchen > Faucets" |

### 五点描述变量

| 变量 | 格式 | 取值说明 |
|------|------|---------|
| `{bullet}` | 文字 | 单条 Bullet Point 内容（150-255字符） |
| `{bullet_1}` ~ `{bullet_5}` | 文字 | 第1~5条 Bullet Point 内容 |

### Review 分析变量

| 变量 | 格式 | 取值说明 |
|------|------|---------|
| `{p5}` ~ `{p1}` | 数字，保留1位 | 5星~1星的百分比，如 62.3 |
| `{p}` | 文字 | 正面高频词（逗号分隔） |
| `{n}` | 文字 | 负面高频词（逗号分隔） |
| `{root_cause_analysis}` | 文字 | 差评根因分析（3-5句话） |

### CDQ 变量

| 变量 | 格式 | 取值说明 |
|------|------|---------|
| `{attr}` | 整数 1-10 | CDQ-结构化属性分 = 维度7得分 |
| `{image}` | 整数 1-10 | CDQ-图片质量分 = 维度8得分 |
| `{other}` | 数字，保留1位 | CDQ-其他内容分 = (维度4 + 维度5) / 2 |
| `{passed}` | 整数 | CDQ 红线项通过数量 |
| `{total}` | 整数 | CDQ 红线项总数量（上下文：CDQ 区段） |
| `{redline_note}` | 文字 | 红线项总结，如 "全部通过" / "2项未通过：标题与属性不一致" |

### 实体与属性变量

| 变量 | 格式 | 取值说明 |
|------|------|---------|
| `{completeness}` | 整数 0-100 | 后台属性填写率百分比 |
| `{v}` | 文字 | 属性当前值，未填写写"—" |
| `{source}` | 文字 | 数据来源，如 "标题" / "Bullet" / "后台属性" / "A+" |
| `{missing_entities}` | 列表 | 缺失的实体维度，逗号分隔，无则写"无" |
| `{entity_analysis}` | 文字 | 实体覆盖分析（3-5句话） |

### 图片与视频变量

| 变量 | 格式 | 取值说明 |
|------|------|---------|
| `{desc}` | 文字 | 图片内容描述（≤20字） |
| `{issue}` | 文字 | 图片问题描述，无问题写"无" |
| `{video}` | 整数 1-10 | 视频分析评分，无视频写"N/A" |
| `{video_analysis}` | 文字 | 视频分析摘要，无视频写"无视频" |

### 竞品变量

| 变量 | 格式 | 取值说明 |
|------|------|---------|
| `{advantages}` | 文字 | 可借鉴的竞品优势（3-5条） |
| `{pricing_matrix}` | 表格 | 价格对比矩阵（markdown 表格） |

### AI 相关变量

| `{alexa_readiness}` | 文字 | 强/中/弱，基于实体完整度判定 |
| `{ai_understood}` | 文字 | AI 已能理解的信息（列表形式） |
| `{ai_cannot}` | 文字 | AI 可能答不好的问题（列表形式） |
| `{ai_not_recommend}` | 文字 | AI 可能不优先推荐的原因（列表形式） |
| `{risk_level}` | 文字 | 高（≥3项风险）/ 中（1-2项）/ 低（0项） |
| `{r}` | 文字 | 单项风险结果，如 "存在" / "不存在" |

### 行动计划变量

| `{dim}` | 文字 | 对应维度编号，如 "维度1-标题" |
| `{actions}` | 列表 | 同一优先级下的多个行动项，逗号或换行分隔 |

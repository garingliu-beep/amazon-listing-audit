## 关键词布局优先级

1. **标题（最高优先，75字符）**：品牌 + 核心产品词 + 1-2 个关键差异化属性
2. **Product Highlights（125字符）**：次级关键词 + 材质/功能/规格/场景/人群（不重复标题词）
3. **五点描述（5条×500-1000字符）**：功能卖点词 + 技术参数词 + 用户痛点词
4. **后台搜索词（250字节）**：长尾词 + 同义词 + 西班牙语/德语词 + 拼写变体 + 缩写
5. **后台其他字段**：Target Audience / Subject Matter / Intended Use / Other Attributes
6. **产品描述/A+**：场景词 + 品牌故事词 + 差异化词（A+ 文本不被搜索索引）

**75+125 重新分配框架：** 标题从 200→75 字符后，多余关键词应迁移到 Product Highlights（125字符），两者都被搜索索引。核心原则是重新分配，不是删除。

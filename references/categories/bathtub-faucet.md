## 4. 专属合规要求

### 水相关认证（美国站）
- [ ] Lead-Free 合规（NSF/ANSI 61 或 NSF 372）— **必查**
- [ ] CUPC/cUPC 认证（如适用）
- [ ] 流量标注准确（浴缸流量通常 4-8 GPM，远高于面盆）

### 澳大利亚站
- [ ] WELS 评级
- [ ] WaterMark 认证

### 特别注意
- 浴缸水龙头流量通常**不适用** WaterSense（WaterSense 主要针对面盆和花洒）
- Freestanding 安装需注意地面管道适配说明

- 同 Installation Type（Freestanding vs Freestanding，差异大）

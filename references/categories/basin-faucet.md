## 4. 专属合规要求

> 与厨房水龙头高度重叠。

### 水相关认证（美国站）
- [ ] Lead-Free 合规（NSF/ANSI 61 或 NSF 372）— **必查**
- [ ] WaterSense 节水认证（1.0 GPM 以下建议申请）
- [ ] CUPC/cUPC 认证（如适用）
- [ ] ADA 合规（如有无障碍要求）

### 澳大利亚站
- [ ] WELS 评级
- [ ] WaterMark 认证

## 4. 专属合规要求

### 联邦/州法规（美国站）
- [ ] **联邦流量上限 2.5 GPM**（所有花洒）— **必查**
- [ ] 部分州更低：科罗拉多 2.0 GPM，加州 1.8 GPM
- [ ] WaterSense 认证（≤2.0 GPM + 满足冲洗性能标准）
- [ ] Lead-Free 合规（NSF/ANSI 61）— **必查**
- [ ] CUPC 认证（如适用）

### 澳大利亚站
- [ ] WELS 评级（星级 + 流量标注）
- [ ] WaterMark 认证

### 特别注意
- **流量是花洒品类的核心合规项**，超标可能被下架
- 花洒面板（Shower Panel）可能涉及电气安全（LED/温度显示），需额外检查

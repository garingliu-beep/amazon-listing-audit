# 合规性检查清单

## 一、标题合规(2026年7月27日起全球执行)

- [ ] 字符数 ≤ 75(含空格),超过将被 AI 自动建议更新
- [ ] 首字母大写(非全大写)
- [ ] 不使用特殊符号
- [ ] 不包含促销信息(Free Shipping / Best Seller / Hot Sale)
- [ ] 不包含主观评价词(Best / Top / Amazing)
- [ ] 不包含价格信息
- [ ] 不包含卖家信息
- [ ] 不使用全大写(除缩写如 USB、LED)
- [ ] 品牌名与注册品牌一致
- [ ] 同一关键词原则上不重复两次以上
- [ ] 用最少信息准确描述商品

## 二、Product Highlights 合规(2026年7月27日起新增)

- [ ] 字符数 ≤ 125
- [ ] 逗号分隔短语形式(不使用完整句子)
- [ ] 不重复标题中已有的词
- [ ] 覆盖材质/功能/场景/人群/规格
- [ ] 不包含促销信息、卖家信息、主观评价词

## 三、图片合规

**主图:**
- [ ] 纯白背景(RGB 255,255,255),无阴影/渐变/边框
- [ ] 产品占比 ≥ 85%
- [ ] 最长边 ≥ 1000px(推荐 2000px+)
- [ ] 最大文件大小 10 MB
- [ ] 格式:JPEG / PNG / TIFF / GIF(非动画)
- [ ] 颜色模式:sRGB
- [ ] 宽高比:1:1(正方形) 推荐
- [ ] 无文字/标志/水印/边框
- [ ] 展示实际产品
- [ ] 完整展示产品(不裁剪边缘)

**副图(最多 9 张):**
- [ ] 不含不当内容
- [ ] 不包含保修信息文字
- [ ] 信息图数据准确可验证
- [ ] 不使用对比图贬低竞品
- [ ] 信息图文字清晰可读(Alexa for Shopping 用 OCR 扫描)
- [ ] 推荐图片栈:主图→功能信息图→尺寸图→场景图→细节图→包装图→对比图/社会证明

## 四、五点描述合规

- [ ] 不包含卖家信息
- [ ] 不包含促销信息
- [ ] 不使用 HTML 标签
- [ ] 认证声明真实可验证
- [ ] 不重复标题中完全相同的内容

## 五、产品描述合规

- [ ] 不包含 HTML(A+ 除外)
- [ ] 不包含外部链接
- [ ] 不包含竞品比较
- [ ] 不使用绝对化用语

## 六、后台搜索词合规

**限制为 250 字节(bytes),不是字符!超出 1 字节,亚马逊忽略整个字段。**

- [ ] 不包含竞品品牌名
- [ ] 不包含 ASIN
- [ ] 不使用主观词
- [ ] 不使用不相关热门词
- [ ] 不使用他人商标词
- [ ] 用空格分隔,不用逗号
- [ ] 不使用填充词(a/an/the/for/with)
- [ ] 不重复标题/Bullet/描述中已有的词
- [ ] 包含同义词/拼写错误/缩写/多语言词/长尾变体

**其他可索引后台字段(独立于 250 字节限制):**
- Target Audience / Subject Matter / Other Attributes / Intended Use

## 七、后台属性合规

- [ ] Material 属性与实际一致
- [ ] 认证属性与实际一致
- [ ] 尺寸/规格数据准确
- [ ] 使用 Amazon 系统标准选项值
- [ ] 所有必填属性已填写

## 八、产品安全与认证(卫浴水龙头)

- [ ] Lead-Free 合规(NSF/ANSI 61 或 NSF 372)
- [ ] CUPC 认证(如适用)
- [ ] 真实材质声明
- [ ] 节水声明有认证支撑
- [ ] Prop 65 警告(加州站必须)

## 九、知识产权合规

- [ ] 图片原创或有授权
- [ ] 不使用竞品品牌名或商标
- [ ] 不复制竞品 A+ 设计
- [ ] 品牌名不与已有品牌过于相似

## 十、QA 与 Review 合规(Alexa for Shopping 时代)

**QA 优先级高于标题:AI 抓取 QA 的优先级高于标题,是获取语义流量的核心入口。**

- [ ] QA 区域已主动布局高频场景化问题
- [ ] QA 问题已精准作答
- [ ] Review 中的高频正面词已融入 Listing 卖点
- [ ] Review 中的高频负面词已在 Listing/QA 中主动回应
- [ ] 无刷评行为
- [ ] 通过 Amazon Vine 等官方渠道获取初始评论
- [ ] Review 邀请引导买家分享具体使用场景

## 十一、变体合规

- [ ] 变体关系正确
- [ ] 变体属性准确
- [ ] 不创建"僵尸变体"
- [ ] 变体间差异明确标注

## 十三、CDQ 算法合规（2026年新增）

**CDQ(Content Data Quality)取代 IDQ,评分直接决定自然流量分配。**

**CDQ 权重:** 结构化属性 30% | 标题质量 25% | 五点描述 15% | 图片质量 15% | 其他内容 15%

**CDQ 红线项(任何一项未通过则 CDQ = min(加权分, 6.0)):**
- [ ] 标题内容与后台属性一致
- [ ] 变体命名统一规范
- [ ] 核心参数真实准确,无夸大
- [ ] 品类核心属性填写率 ≥ 50%
- [ ] 属性值使用 Amazon 标准选项
- [ ] 标题/Bullet/描述/属性之间无矛盾信息

## 十四、AI 自动改写风险（2026年7月27日起）

**超过 75 字符的标题将被 AI 自动替换。品牌卖家有 14 天审核窗口,无品牌卖家无否决权。**

**卖家对 AI 生成内容负全责。** 14 天不操作 = 被动批准。

**75+125 重新分配框架:**
- Zone 1 - 标题(75c):品牌 + 最高搜索量产品词 + 1-2 个关键差异化属性
- Zone 2 - Item Highlights(125c):次级关键词 + 材质/功能/规格/场景/人群
- 核心原则:重新分配,不是删除。

**风险检查项:**
- [ ] 标题中是否有易被 AI 删除的限定词
- [ ] 是否有品牌特有术语可能被通用词替换
- [ ] 是否有合规声明可能被 AI 删除
- [ ] 是否有风格/定位词可能被 AI 偏移

**品牌卖家:** 每周巡检"审核商品信息更改"页面
**无品牌卖家:** 7 月 27 日前主动优化标题至 ≤75 字符

- [ ] 水压要求已标注
- [ ] 水管接口标准已标注
- [ ] 适用场景已明确
- [ ] 不兼容情况已明确

## 十六、表现信号评估（2026年新增，维度14）

**本维度需要卖家后台数据支持。如无法获取,按默认中等分数(5/10)处理。**

**A10 算法排名因素权重:**
- 销售速度:35-40%
- 相关性与关键词:25-30%
- 客户满意度:20-25%

**核心指标清单:**
- [ ] 转化率(CVR):目标 >10%,优秀 >15%
- [ ] 点击率(CTR):目标 >3%,优秀 >5%
- [ ] 销售速度:有机销售 > PPC 销售
- [ ] Review 速度:月增 5+ 条
- [ ] 退货率:目标 <5%,优秀 <3%
- [ ] BSR 趋势:稳定或上升
- [ ] 外部流量:社交媒体/邮件/网站引流
- [ ] 卖家权威:ODR <1%、迟发率 <4%

**评分标准:**
| 分数 | 转化率 | CTR | Review月增 | 退货率 | BSR |
|------|--------|-----|-----------|--------|-----|
| 9-10 | >15% | >5% | 10+ | <3% | 稳定上升 |
| 7-8 | 10-15% | 3-5% | 5-10 | <5% | 稳定 |
| 5-6 | 7-10% | 1-3% | 1-5 | 5-8% | 波动 |
| 3-4 | <7% | <1% | 几乎无 | >8% | 下降 |
| 1-2 | 极低 | 极低 | 无 | 极高 | 被抑制 |

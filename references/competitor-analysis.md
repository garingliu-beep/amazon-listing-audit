# 竞品深度分析方法论

## 竞品选择标准

### 来源优先级
1. **Listing 页面"相似产品"**：Customers who viewed this item also viewed / Similar items
2. **类目 Top 20**：同类目 BSR 排名前 20 的产品
3. **搜索结果 Top 10**：核心关键词搜索结果前 10 名
4. **用户指定**：用户手动提供的竞品 ASIN

### 筛选条件（至少满足 3 项）
- 同产品类型（如都是单孔面盆水龙头）
- 同安装方式（如都是 deck mount）
- 同价位段（±30% 价格区间）
- 同材质（如都是 brass / metal）
- 同风格（如都是 modern / contemporary）

## 竞品分析维度

### 1. 标题分析
| 检查项 | 分析方法 |
|--------|---------|
| 字符数 | 是否 ≤75 字符 |
| 核心词位置 | 核心产品词在前 40 字符内？ |
| 关键词覆盖 | 提取标题中的所有关键词 |
| 结构 | 品牌+产品+属性 的排列方式 |
| 差异化 | 竞品标题中有而本产品没有的关键词 |

### 2. Item Highlights 分析（2026年7月27日起）
| 检查项 | 分析方法 |
|--------|---------|
| 是否已填写 | 7月27日后为重要搜索索引字段 |
| 字符利用 | 是否充分利用 125 字符 |
| 内容覆盖 | 覆盖了哪些维度（材质/功能/场景/人群/规格） |
| 与标题重复度 | 是否有关键词浪费（与标题重复） |
| 短语格式 | 是否为逗号分隔短语（非完整句子） |

### 3. Bullet Points 分析
| 检查项 | 分析方法 |
|--------|---------|
| 卖点覆盖 | 每条 bullet 聚焦什么卖点 |
| 关键词嵌入 | 是否自然嵌入搜索关键词 |
| 参数数据 | 是否包含具体数据（GPM、尺寸、认证） |
| 差异化卖点 | 竞品强调了什么本产品没有的卖点 |
| 格式 | 是否使用【】/大写等格式增强可读性 |

### 4. 图片策略分析
| 检查项 | 分析方法 |
|--------|---------|
| 图片数量 | 总图片数 |
| 主图风格 | 纯白/场景/角度 |
| 信息图类型 | 材质图/尺寸图/对比图/安装图 |
| 场景图 | 是否有 lifestyle 场景图 |
| 视频 | 是否有产品视频 |
| A+ 图片 | A+ 内容的丰富度 |

### 5. 价格与促销
| 检查项 | 分析方法 |
|--------|---------|
| 当前价格 | 具体价格 |
| 原价/List Price | 是否有划线价 |
| Coupon | 是否有绿色 Coupon 标签 |
| Deal | 是否参与 Lightning Deal / Best Deal |
| 变体价格差 | 不同变体间的价格差异 |

### 6. 评分与 Review
| 检查项 | 分析方法 |
|--------|---------|
| Rating | 星级评分 |
| Review 数量 | 总评论数 |
| 评分分布 | 5星/4星/3星/2星/1星占比 |
| Review 高频词 | 正面/负面高频关键词 |
| Vine | 是否有 Vine Review |

### 7. 变体策略
| 检查项 | 分析方法 |
|--------|---------|
| 变体数量 | 总变体数 |
| 变体类型 | 颜色/尺寸/包装组合 |
| 热门变体 | 哪个变体 Review 最多 |
| 缺失变体 | 本产品应该有但没有的变体 |

### 8. 属性完整度
| 检查项 | 分析方法 |
|--------|---------|
| 可见属性 | 从页面可见的 Material/Style 等 |
| 参数完整度 | Technical Specifications 区域的完整度 |
| 与本产品对比 | 竞品有而本产品缺失的属性 |

## 可借鉴的竞品优势提取

1. **标题结构借鉴**：竞品标题中高效的关键词排列方式
2. **卖点差异化**：竞品强调但本产品缺失的卖点（如 pop-up drain、deck plate）
3. **图片策略借鉴**：竞品的信息图类型、场景图风格
4. **定价策略借鉴**：竞品的促销手段、价格定位
5. **Review 策略借鉴**：竞品的 Vine 计划、差评回应方式

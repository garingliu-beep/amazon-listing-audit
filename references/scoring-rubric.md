# 评分标准（满分 10 分制）

## 评分理念

基于亚马逊 AI 搜索时代逻辑：评估 Listing 的商品实体完整度和 AI 可理解性。

---

## 1. 标题优化（权重 11%）

**标题字符上限：75 字符（含空格）— 2026年7月27日起全球执行**

**⚠️ 前台显示：** 标题与 Product Highlights 拼接显示，系统自动插入 `|` 分隔符。买家看到的是 `{Title} | {Highlights}` 的连续文本。优化标题时需考虑拼接后的整体可读性，避免关键词在标题和 Highlights 间重复。

| 分数 | 标准 |
|------|------|
| 9-10 | 核心关键词前置（前40字符），品牌名合理，≤75字符，实体化表达，可读性强 |
| 7-8 | 关键词基本覆盖，品牌名位置可接受，≤75字符，少量属性缺失 |
| 5-6 | 关键词覆盖不足或位置靠后，>75字符或<40字符，属性词较少 |
| 3-4 | 关键词堆砌严重或严重缺失，可读性差 |
| 1-2 | 几乎无有效关键词，或违反亚马逊标题规范 |

---

## 2. Product Highlights（权重 7%）

**新增模块（2026年7月27日起）：125 字符，逗号分隔短语，可被搜索索引**

**⚠️ 前台显示：** Highlights 紧跟在标题之后，与标题拼接显示为 `{Title} | {Highlights}`。因此：
- 不得重复标题中已有的关键词（浪费索引空间 + 买家看到重复词困惑）
- 需考虑与标题拼接后的整体流畅度和可读性
- Highlights 应补充标题未覆盖的信息，而非复述

| 分数 | 标准 |
|------|------|
| 9-10 | 充分利用125字符，覆盖材质/功能/场景/人群/规格，短语形式 |
| 7-8 | 覆盖4个以上维度，短语形式，关键词相关性高 |
| 5-6 | 覆盖2-3个维度，或使用完整句子，或字符利用不足 |
| 3-4 | 仅覆盖1个维度，或与标题高度重复 |
| 1-2 | 未填写或无意义内容 |

---

## 3. 五点描述（权重 9%）

| 分数 | 标准 |
|------|------|
| 9-10 | 5条完整，每条以 Benefit Label 开头回答一个买家问题，嵌入关键词，含具体参数/数据 |
| 7-8 | 5条完整，卖点覆盖较好，关键词自然，部分缺数据 |
| 5-6 | 不足5条或部分过短（<100字符），卖点重复或覆盖不全 |
| 3-4 | 严重不足，缺乏卖点逻辑 |
| 1-2 | 无 bullet points 或无意义 |

**写法标准：** 每条 bullet 以 Benefit Label 开头，回答一个具体买家问题，含参数数据，嵌入1-2个关键词。系统允许范围 500-1000 字符/条，推荐实践 150-255 字符/条（简洁高效，避免冗余）。

---

## 4. 产品描述 / A+ Content（权重 5%）

| 分数 | 标准 |
|------|------|
| 9-10 | A+ 完整，含品牌故事/对比表/安装指南/FAQ，图文并茂 |
| 7-8 | 有 A+，内容完整但设计一般，或缺部分模块 |
| 5-6 | 仅基础文本无 A+，或 A+ 较简单 |
| 3-4 | 描述不完整或无 A+ |
| 1-2 | 无产品描述 |

**重要：** A+ 文本不被搜索索引（但图片 alt text 可能被索引）。Alexa for Shopping 会读取 A+ 中的品牌故事模块和 FAQ 模块。

---

## 5. 后台搜索词（权重 5%）

**限制为 250 字节（bytes），不是字符！超出 1 字节，亚马逊将忽略整个字段。**

| 分数 | 标准 |
|------|------|
| 9-10 | 填满250字节，含标题未覆盖的长尾词/同义词/拼写变体/西语词，无重复，同时填写其他可索引后台字段 |
| 7-8 | 基本填满，关键词相关性高，少量遗漏 |
| 5-6 | 填充不足50%，或含较多无关词 |
| 3-4 | 严重不足或关键词选择不当 |
| 1-2 | 未填写或完全无关 |

---

## 6. 实体 SEO 与关键词（权重 9%）

**评估 Listing 作为"商品实体"的完整度。**

| 分数 | 标准 |
|------|------|
| 9-10 | 五维全覆盖（产品类型/功能属性/适用对象/适用场景/尺寸规格），语义丰富 |
| 7-8 | 覆盖4个以上维度，语义自然 |
| 5-6 | 覆盖2-3个维度，有明显遗漏 |
| 3-4 | 仅覆盖产品类型，缺少属性/场景/人群 |
| 1-2 | 几乎无实体构建，纯关键词堆砌 |

---

## 7. 后台属性完整性（权重 9%）

**后台属性是 AI 系统理解产品的核心数据源，直接影响 CDQ 评分**

| 分数 | 标准 |
|------|------|
| 9-10 | 必填+选填全部填写，数据准确，使用标准化值 |
| 7-8 | 必填全部填写，选填覆盖>70% |
| 5-6 | 必填基本填写，选填覆盖40-70% |
| 3-4 | 必填有遗漏，选填<40% |
| 1-2 | 大量未填写 |

---

## 8. 图片检查（权重 12%）

| 分数 | 标准 |
|------|------|
| 9-10 | 主图合规+7-9张图覆盖主图/角度/细节/尺寸/场景/信息图/包装图，信息图文字清晰可读 |
| 7-8 | 主图合规，6-7张，覆盖主要类型 |
| 5-6 | 主图基本合规，4-5张，缺信息图或场景图 |
| 3-4 | 不足4张，主图有问题 |
| 1-2 | 严重不足或主图不合规 |

**附加：视频分析评分影响本维度**
- 有专业视频（安装演示+场景+字幕+30-90s）→ +1-2 分（上限10）
- 有视频但内容简单 → +0.5 分
- 无视频 → 不加分

---

## 9. 定价与促销策略（权重 5%）

**Alexa for Shopping 时代：频繁无规律调价被 AI 判定为"低信任度"，减少推荐权重。**

| 分数 | 标准 |
|------|------|
| 9-10 | 价格在类目合理区间，有 Coupon/Deal，变体价格梯度合理，价格稳定 |
| 7-8 | 价格合理，有基础促销，价格基本稳定 |
| 5-6 | 价格偏高或偏低，无促销，或有频繁调价记录 |
| 3-4 | 价格明显不合理，或价格波动频繁被 AI 标记 |
| 1-2 | 定价严重偏离市场 |

---

## 10. Review 与评分分析（权重 7%）

**Alexa 具备虚假评价识别能力，刷单不仅降权还可能导致封号。QA 优先级高于标题。**

| 分数 | 标准 |
|------|------|
| 9-10 | 评分4.3+，评论数高于类目中位数，差评已回应，有 Vine，QA 已主动布局 |
| 7-8 | 评分4.0+，评论数中等，差评基本回应，QA 有基础覆盖 |
| 5-6 | 评分3.5-4.0 或评论数偏少，QA 覆盖不足 |
| 3-4 | 评分<3.5 或评论严重不足，QA 几乎空白 |
| 1-2 | 评分极低或无评论 |

**附加：Review 深度分析评分影响本维度**
- 差评高频问题100%回应 + Highlights完全一致 → +1-2 分（上限10）
- 70%+回应 → +1 分

---

## 11. 竞品对比（权重 3%）

| 分数 | 标准 |
|------|------|
| 9-10 | 关键维度均优于或持平竞品 |
| 7-8 | 多数维度持平，个别有优势 |
| 5-6 | 部分维度落后 |
| 3-4 | 多数维度明显落后 |
| 1-2 | 全面落后 |

---

## 12. 类目与节点选择（权重 3%）

| 分数 | 标准 |
|------|------|
| 9-10 | 叶节点最精准，Browse Node 正确 |
| 7-8 | 基本正确，可更精准 |
| 5-6 | 可接受但不是最优 |
| 3-4 | 有明显偏差 |
| 1-2 | 类目错误 |

---

## 13. 合规性检查（权重 5%）

| 分数 | 标准 |
|------|------|
| 9-10 | 无任何合规问题 |
| 7-8 | 极少量轻微问题 |
| 5-6 | 1-2个需关注的合规问题 |
| 3-4 | 多个合规风险 |
| 1-2 | 严重违规可能导致下架 |

**附加：限制与兼容性检查影响本维度**
- 所有兼容性项已明确标注 → +0.5 分（上限10）

---

## 14. 表现信号（权重 10%）

**本维度需要卖家后台数据支持。如无法获取，按默认中等分数（5/10）处理。**

| 分数 | 标准 |
|------|------|
| 9-10 | 转化率 >15%，CTR >5%，Review 月增 10+，退货率 <3%，BSR 稳定上升 |
| 7-8 | 转化率 10-15%，CTR 3-5%，Review 月增 5-10，退货率 <5%，BSR 稳定 |
| 5-6 | 转化率 7-10%，CTR 1-3%，Review 月增 1-5，退货率 5-8%，BSR 波动 |
| 3-4 | 转化率 <7%，CTR <1%，Review 几乎无增长，退货率 >8%，BSR 下降 |
| 1-2 | 转化率极低，Listing 被抑制或暂停 |

**核心指标：**
- 转化率（CVR）：A10 最重要单一变量
- 点击率（CTR）：主图+标题决定
- 销售速度：有机销售 > PPC 销售
- Review 速度和近期度
- 外部流量
- 退货率
- 卖家权威

---

## CDQ 评分计算方法

**CDQ = 结构化属性×30% + 标题质量×25% + 五点描述×15% + 图片质量×15% + 其他内容×15%**

- 结构化属性 = 维度7得分
- 标题质量 = 维度1得分
- 五点描述 = 维度3得分
- 图片质量 = 维度8得分
- 其他内容 = (维度4 + 维度5) / 2

**红线规则：** 红线项任何一项未通过时，CDQ = min(加权平均分, 6.0)。
